Professional Self-Assessment

Throughout my time in the Computer Science program at Southern New Hampshire University, I have developed a strong foundation in software development, data structures, database systems, and system design. The completion of this ePortfolio reflects both my technical growth and my ability to apply these concepts to real-world problems.

One of the most significant areas of growth has been my ability to design and improve software systems. Through the rescue animal management system, I transitioned a basic console-based application into a more advanced system by implementing a graphical user interface, improving code structure, and incorporating input validation.

I strengthened my understanding of data structures and algorithms by optimizing performance using HashMaps, improving lookup operations from O(n) to O(1). Additionally, I implemented a file-based persistence system to simulate database functionality, enabling full CRUD operations and improving data reliability.

These artifacts demonstrate my ability to evaluate, enhance, and implement computing solutions that are efficient, scalable, and aligned with real-world industry practices.
