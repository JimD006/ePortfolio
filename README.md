# ePortfolio

My name is James Webb, and this ePortfolio showcases the work I completed during my Computer Science program at Southern New Hampshire University. This portfolio highlights my growth in software development, data structures, database management, and system design through the enhancement of a core project.

The primary artifact featured in this portfolio is the Rescue Animal Management System, which was originally developed as a console-based application and later enhanced as part of my capstone. Through this process, I transformed the system by improving usability, performance, and data management. Key enhancements include the implementation of a graphical user interface using Java Swing, optimization of data retrieval using HashMaps to achieve O(1) lookup performance, and the addition of a file-based persistence layer to simulate database functionality with full CRUD operations.

These improvements demonstrate my ability to analyze an existing system, identify limitations, and apply appropriate software engineering and computer science principles to enhance functionality and efficiency. This portfolio reflects not only my technical skills, but also my ability to design scalable solutions and adapt systems to meet evolving requirements.

In addition to coursework, my background in IT operations, database analysis, and security has strengthened my ability to approach technical challenges from both a development and operational perspective. This portfolio represents my readiness to contribute to real-world software development and database environments.
